Traffic Racer

MIPS

This project was part of my coursework for CSCB58 - Computer Organization at the University of Toronto.

This was a solo project built in Assembly. We used the MARS Assembly Simulator (based on MIPS-32 architecture) for the coursework in this class. The end goal of this project was to build a simple functioning 2D game using keyboard inputs and a bitmap display.

The features I successfully implemented included: random car spawns (speeds, lanes) using a psuedo-random number generator, multiple speeds for the player car, a live score bar, a lives remaining bar and hard mode after a specific amount of time had passed.
