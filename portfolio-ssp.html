Software Design Project

An Android App

This project was part of my coursework for CSCB07 - Software Design at the University of Toronto.

In a group of 6, we used the Agile software development cycle to build an Android application satisfying 7 user stories. The app was developed using Java, with a Firebase database storing the data. The app was intended to support sports venues, so that regular members could schedule and join events at venues, and administrators could add venues.

My responsibilities for this project included the following: adding support for users to join events, fixing bugs, analyzing the user stories and designing the schema for the data. We completed all the tasks within 2 weeks.
